# Open Source Contribution Tracker

## Project Overview

The Open Source Contribution Tracker is a web application that tracks GitHub users and repositories using GitHub REST API.

The project displays:
- User profiles
- Repository details
- Stars and forks
- Open source activity


# Features

- GitHub user search
- Repository tracking
- GitHub API integration
- Responsive UI

# Technologies Used

- Python
- Flask
- GitHub API
- HTML/CSS


# Installation

Install required libraries:

pip install flask requests


# Run Project

python app.py


# GitHub API

https://api.github.com/users/{username}
